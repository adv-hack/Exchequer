ESN Resynchronisation To resynchronise Enterprise Site Numbers, where companies attached to the Multi Company Manager display different ESN's.  These instructions are only valid for Enterprise versions from 4.30c to 4.32.001.  The resynchronisation program must only be used with no users accessing Exchequer, Enterprise or the OLE link to Enterprise, including users of bespoke systems using the Com Toolkit or DLL Toolkit.  This procedure requires the use of the Enterprise Daily System Password issued by Exchequer Software or retrieved from this site.  The resynchronisation program requires a Resync Companies Password obtained from Exchequer Support or from this site. If it is necessary to contact Exchequer Software please be prepared to provide the following information:         The version of Enterprise,        The version of the Multi Company Manager (Help, About from the MCM),        The ESN from the Multi Company Manager,        The ESN’s of all companies listed in the Multi Company Manager,  The resynchronisation command, when run, replicates the ESN belonging to the Main Enterprise Company to all companies including the MCM ESN if it is different.   Running the Resynchronisation command: 1.From a “DOS prompt”, (Start, Program Files, DOS Prompt), or Command prompt (Start, Run, Cmd),  2.Locate the main Enterprise directory,  3.Enter the following command exactly as shown and press Return. Note: RSYNC must be entered in uppercase,  entrprse /RSYNC:xxxxx  (Where xxxxx = the Resync Companies Password supplied)  4.This returns the MCM, 5.From “File”, Choose “Login” from the dropdown list, Enter system password and click on “OK”, From “File”, Choose “synchronise companies”.  6.Note: If the “synchronise” option is not available from the 'File' dropdown menu, the 'entrprse' command was probably not entered correctly. Try repeating the command from the DOS prompt.  7. When run successfully, the ESN' are updated immediately.  8. Close MCM and exit from DOS screen. 9.Finally from the MCM, versions 4.31.004 and above, login as system user and confirm all companies display the same ESN from the MCM. For versions prior to 4.31.004, each Enterprise company must be opened to locate the ESN from Help, About.
