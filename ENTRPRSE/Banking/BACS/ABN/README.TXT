BACS Replacement Hook - Delphi 5.0 Customisation Example
========================================================

This example hook written for the Delphi 5.0 Customisation system replaces
the standard BACS export in Exchequer Enterprise with an export suitable
for the system ran by ABN, a large Dutch bank.

See the online help, EntCustm.Hlp, for details of the BACS Export Hooks.
