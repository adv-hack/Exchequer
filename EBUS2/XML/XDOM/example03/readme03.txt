FILE DESCRIPTION

This example source code shows how to load
and  display  the source  of XML files and 
their  entities   and  parameter  entities
using  the XDOM  Delphi unit.  In order to
have  it  run,  XDOM version 3.6  or above 
must be installed  on your  Delphi system. 
The  latest XDOM version  is available  at 
"http://www.philo.de/xml/" free of charge.
