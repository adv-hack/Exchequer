TFrameViewer, ThtmlViewer, and TFrameBrowser

The HTML component set consists of the ThtmlViewer, TFrameViewer, and TFrameBrowser components. All three are HTML document display components:

ThtmlViewer The basic component. ThtmlViewer displays single (non-frame) documents. It also forms the basis for the other two components.
TFrameViewer Displays both Frame and single HTML documents. TFrameViewer is oriented more for local disk file use.
TFrameBrowser    Also displays Frame and single HTML documents. However, TFrameBrower is oriented toward Internet style protocols and URL usage. Additional code and/or components are generally required to use TFrameBrowser.

If you know that your appication will not be using Frames, you can save a fair amount of code overhead by using ThtmlViewer rather than one of the Frame components. Many operations are also simpler using ThtmlViewer.

Version 8 Compile Time Options

Version 8 has several possible compile time options controlled by defining symbols. You can find these in the HTMLCONS.INC file. None of these symbols are initially defined. To make any one (or all) of these symbols active, remove the "." from the {$.Define XXXX} sequence and recompile the code.

The possibilities are:

NoGIF

Defining NoGIF disables GIF imaging capabilities. Other image types are unaffected. NoGIF may be used with or without the graphics DLL.

UseDLL

Define UseDLL to use the optional graphics DLL and obtain the extra graphics capabilities it provides.

The Former NoDLL compile time option is no longer required as not using the DLL is the default condition.

The Former NoUnderline and PrintTableBackground compile time options are now handled by the htOptions and fvOptions properties.

Graphics Capabilities and the Optional Graphics DLL

The table below indicates the graphics capabilities of the various versions of the HTML components when used with and without the optional graphics DLL:

Type Without Graphics DLL With Graphics DLL
Bitmap Delphi 1-5 Delphi 1-5
GIF Delphi 1-5 Delphi 1-5
Animated GIF Delphi 1-5 Delphi 1-5
JPEG Delphi 3, 4, 5 Delphi 1-5
PNG None Delphi 1-5

The optional graphics DLLs used with the HTML components are named as follows:

  ILDA16.DLL Delphi 1 Professional version
ILDA32.DLL 32 bit Professional versions
ILDA16T.DLL Delphi 1 demo

To use the DLL, define UseDLL before compiling your application.  The DLL may be located in the either the Windows\System directory or the same directory as your program. Be sure and supply the appropriate DLL with your application.

Mouseover Images

A Mouseover Image is one that changes when the mouse passes over it. The HTML components use a special syntax to implement mouseover images. It's done with the <img> tag as follows:

  • the SRC="..." attribute should specify an animated GIF image
  • Add an Active attribute

With the Active attribute, the GIF will normally just display the first frame. When the mouse passes over, it will display the second frame if there are 2 frames or will animate if there are more than 2 frames.
